Steve does PowerShell

  • Large amounts of data in an array

  • Clip
  • VS Code Regex
  • Quickly converting a list into a PowerShell Array
  • Start of a line: ^
  • End of a line: $
  • Blank Lines: ^\s*$\n
  • Line break: \n
  • Find an email address (Warning: May grab apostrophe characters inadvertently.): [a-z0-9!#\$%&'*+/=?^_`{|}~-]+(?:\.[a-z0-9!#\$%&'*+/=?^_`{|}~-]+)*@(?:[a-z0-9](?:[a-z0-9-]*[a-z0-9])?\.)+[a-z0-9](?:[a-z0-9-]*[a-z0-9])?
  • Taking a list of names/etc and converting to an array: Find \n and replace with "`\n," (yes, including double quotes) then add $MyArray = @(" to the start of the first line, and ") at the end of the last line.

Ransomware Protection: Controlled Folder Access and FSRM

  • CFA is best for client-side protection (Win10 1709 and up)
  • CFA can be overly annoying at first, whitelisting is recommended
  • CFA hooks into Defender ATP
  • CFA has PowerShell whitelisting and enabling/disabling (or via GPO)
  • Computer configuration ->Administrative templates > Windows components. Next click Windows Defender Antivirus > Windows Defender Exploit Guard > Controlled folder access. There are 3 settings available.

  • FSRM is best for server-side file share protection
