Win 11

  • Vista 2.0? Not really
  • Windows 11 WSL 2 is almost as quick as running Linux natively
  • Installing Wine using Homebrew on macOS: GitHub repo
  • This below will only install Wine Crossover 5.0 (very old version of Wine, but it runs 32-bit apps whereas newer versions of Wine do not + have huge problems with macOS):
brew tap gcenx/wine
brew install --cask --no-quarantine wine-crossover
#To test it, run Explorer:
wine64 explorer
#To uninstall it, run this:
brew uninstall wine-crossover
#If you want to reinstall it:
brew reinstall wine-crossover
  • Newer versions of Wine (these don't really work on newer macOS, but does work on MacOS High Sierra at least, and will install at least on Big Sur/Intel)
brew tap gcenx/wine
brew install --cask --no-quarantine gcenx-wine-stable
#To test it, run Explorer:
wine64 explorer
#OR (your choice of stable vs staging):
brew install --cask --no-quarantine gcenx-wine-staging
#To uninstall (replace xxxxxx with the edition name above):
brew uninstall "gcenx-wine-xxxxxx"

Hope This Helps Tip of the Something

  • Don't sudo rm -rf your docker storage folder
  • Things that may happen:
  • Images gone
  • Errors trying to update containers
  • If a container is restarted it will not come back
  • Portainer will explode
  • You'll have to rebuild all your docker-compose stacks in Portainer (hope you had a backup)
